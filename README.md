A Flutter mobile application developed as part of the Flutter Developer recruitment assignment for Scube Technologies Ltd.  
The application UI and flow are implemented according to the provided Figma design and prototype.

---

## Project Overview

This project focuses on clean Flutter architecture, proper state management, and UI accuracy.

### Features
- Pixel-perfect UI based on Figma
- Clean and readable Flutter code
- Repository Pattern implementation
- Provider for state management
- Dummy / mock data used
- Scalable and maintainable structure

---

## Architecture

The app follows the Repository Pattern to separate UI and data logic.

lib/
│
├── data/
│ ├── models/
│ ├── repositories/
│ └── datasources/
│
├── providers/
│
├── ui/
│ ├── screens/
│ └── widgets/
│
└── main.dart
